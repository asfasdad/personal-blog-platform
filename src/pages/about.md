## About

This is a personal blog built with Astro. The site features a clean, responsive design with support for light and dark modes.

## Features

- Static site generation with Astro
- Markdown/MDX content support
- Tag-based organization
- Responsive design
- Dark/light mode toggle
- Search functionality (via Pagefind)
