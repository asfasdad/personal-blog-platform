## 关于我

我是 Kayden，一个对技术充满热情的开发者。这个博客是我记录技术学习、项目实践和工程思考的地方。

## 关于这个博客

这个博客使用 [Astro](https://astro.build) 框架搭建，部署在 [Cloudflare Workers](https://workers.cloudflare.com) 上。

### 技术栈

- **框架**: Astro 6 (SSR + 静态混合模式)
- **样式**: Tailwind CSS 4
- **部署**: Cloudflare Workers + Pages
- **搜索**: Pagefind (全文搜索)
- **数据库**: Cloudflare D1
- **存储**: Cloudflare R2

### 功能特性

- Markdown 内容支持
- 中英双语切换
- 明暗主题切换
- 全文搜索
- RSS 订阅
- 标签分类
- 管理后台
